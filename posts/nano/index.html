安装nano

zsh自带了nano，但是版本较低，在mac自带的终端安装最新版nano，不要使用iTerm2。

从nano官网复制最新版下载链接，下载nano：

1
2

cd ~/Downloads
wget https://www.nano-editor.org/dist/v6/nano-6.0.tar.gz

解压：

1

tar -xzvf nano-6.0.tar.gz

编译安装：

1
2
3
4

cd nano-6.0
./configure
sudo make
sudo make install

删除安装包：

1
2
3

cd ..
rm nano-6.0.tar.gz
rm -r nano-6.0

查看nano版本是否是最新版：

1

nano -V

新建~/.nanorc文件，写入以下内容：

1
2
3
4
5
6

# highlight syntax
include "/usr/local/share/nano/*.nanorc"
include "/usr/local/share/nano/extra/*.nanorc"

# show line numbers
set linenumbers

nano实用快捷键

  1. 选择多行：按住shift再按上下键
  2. 复制：option + 6
  3. 剪切：ctrl + k
  4. 粘贴从nano复制或剪切的内容：ctrl + u
  5. 撤销：option + u
  6. 重做：option + e
  7. 保存：ctrl + s
  8. 退出：ctrl + x
  9. 跳到行首：ctrl + a
  10. 跳到行尾：ctrl + e
  11. 跳到文件开头：option + \
  12. 跳到文件结尾：option + /
  13. 上一页：ctrl + y
  14. 下一页：ctrl + v
  15. 注释：option + 3
