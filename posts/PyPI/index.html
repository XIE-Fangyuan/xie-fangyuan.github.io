版本号管理

安装bumpver：

1

pip install bumpver

初始化版本，之后在pyproject.toml做适当修改：

1

bumpver init

预览版本号更新：

1

bumpver update --patch --dry

更新版本号：

1

bumpver update --patch

可编辑安装

以可编辑模式从本地源代码安装：

1

pip install -e .

实际上是创建了一个链接。安装之后可以另外创建一个项目，导入该包进行测试，源代码发生修改后可直接测试最新的代码。

构建和发布

安装build和twine：

1

pip install build twine

生成.tar.gz和.wheel文件到dist/：

1

python -m build

检查渲染的PyPI描述页面：

1

twine check dist/*

注册PyPI和TestPyPI账号，分别添加令牌，在$HOME目录下创建.pypirc文件：

1

nano ~/.pypirc

在~/.pypirc中写入：

1
2
3
4
5
6
7

[pypi]
  username = __token__
  password = pypi-xxxxxxxx

[testpypi]
  username = __token__
  password = pypi-xxxxxxxx

发布到TestPyPI

发布Python包到TestPyPI进行测试：

1

twine upload -r testpypi dist/*

从TestPyPI安装：

1

pip install -i https://test.pypi.org/simple/ package-name

卸载Python包：

1

pip uninstall package-name

发布到PyPI

正式发布到PyPI：

1

twine upload dist/*

从PyPI安装：

1

pip install package-name
